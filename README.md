# House of EdTech - React Native (Expo) Assignment

## Overview

A production-style React Native application built with Expo, TypeScript, NativeWind, React Navigation, and React Native Paper.

Inspired by Disney+ Hotstar UI with a scalable architecture, reusable components, and performance optimizations.

## Tech Stack

- Expo SDK 57
- TypeScript
- React Navigation
- NativeWind
- React Native Paper
- React Native Reanimated
- Expo Image

## Features

- Hero Banner
- Horizontal Carousels
- Detail Screen
- Profile Screen
- Mock API Layer
- Skeleton Loading
- Error States
- Dark Mode (Planned)
